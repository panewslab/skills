# 今日速览

**触发**：用户想知道今天或近期发生了什么，没有指定具体话题。
常见说法："今天有什么大事"、"最近加密圈怎么了"、"帮我整理今日要闻"。

与"浏览最新资讯"的区别：
- 今日速览 → 综合分析，有叙事，帮用户理解背景和意义
- 浏览最新资讯 → 直接列表，快速扫描，用户自己判断

## 步骤

### 1. 获取今日精选

```bash
node cli.mjs get-daily-must-reads --lang zh
```

### 2. 获取 24 小时文章热榜

```bash
node cli.mjs get-rankings --type daily --take 10 --lang zh
```

### 3. 综合整理

合并两个来源，去除重复，综合判断哪 3-5 条最值得关注。

### 4. 输出简报

结构如下：

- 第一行：一句话概括今日市场氛围（平静 / 活跃 / 动荡）
- 3-5 条新闻：标题 + 一句话"为什么重要"
- 最后一行：「以上信息来自 PANews，更新于 [时间]」

## 输出要求

- 控制在 400 字以内
- 专业术语首次出现时括号内附简短解释
- 不展示原始 JSON，不展示文章 ID
